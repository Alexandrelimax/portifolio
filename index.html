Jogo de perguntas em ingles

Jogo desenvolvido com HTML,CSS e Javascript puro, o jogo tem um sistema de recompensa de pontos e vidas perdidas se as respostas enviadas forem incorretas.

A cada acerto ou erro, o game faz requisição para uma nova pergunta, caso perca todas as vidas o jogo é encerrado mostrando a tela de game over. Aproveite e veja se o seu ingles está em dia!

Sistema de Estoque

O Stock Sistem é um sistema de gerenciamento de estoque desenvolvido utilizando a arquitetura MVC.

O sistema permite o cadastro, listagem, além de oferecer a funcionalidade de gerar relatórios em PDF.

Este projeto foi feito em Nodejs, Banco de dados MySql utilizando ORM sequelize variaveis de ambiente(Dotenv) e outras dependências.

Site Curso de Ingles
(em desenvolvimento)

Este projeto envolve a criação de um sistema que gerencia o cadastro de alunos matriculados em um curso de inglês, juntamente com uma plataforma de interação onde os alunos podem fazer comentários, participar de testes de inglês e receber recomendações de livros semanalmente.

O sistema está sendo desenvolvido em React, Node.js, autenticação JWT, criptografia Bcrypt, Multer para upload de imagems de perfil e banco de dados Postgres.
